Quick Start Guide

  1. Enter Data: Use the Data tab to enter study data manually, paste JSON, or load a demo dataset.
  2. Configure Analysis: Select effect measure (OR, RR, RD), estimator (REML, DL), and reference treatment.
  3. Run Analysis: Click "Run Analysis" - Guardian will check data quality first.

Statistical Methods

Network Meta-Analysis Framework

NMA Pro uses a graph-theoretical frequentist approach based on the methodology of Rücker (2012) and implemented in the R netmeta package.

Heterogeneity Estimators

EstimatorDescriptionWhen to Use
REMLRestricted Maximum LikelihoodDefault, recommended for most analyses
DLDerSimonian-LairdTraditional method, may underestimate t²
PMPaule-MandelRobust to outliers

Effect Measures

  • OR (Odds Ratio): Ratio of odds between treatments. Use for binary outcomes.
  • RR (Risk Ratio): Ratio of risks. More interpretable than OR.
  • RD (Risk Difference): Absolute difference in risks. Useful for NNT calculations.
  • SMD (Standardized Mean Difference): For continuous outcomes with different scales.

Ranking Methods

P-score: Frequentist analogue of SUCRA. Ranges 0-1, where 1 = best treatment.

SUCRA: Surface Under Cumulative Ranking curve. Equivalent to P-score in frequentist NMA.

Key References

  • Rücker G. Network meta-analysis, electrical networks and graph theory. Res Synth Methods. 2012;3:312-24.
  • Rücker G, Schwarzer G. Ranking treatments in frequentist network meta-analysis. BMC Med Res Methodol. 2015;15:58.
  • Dias S, et al. Network meta-analysis for decision-making. Wiley; 2018.

Interpretation Guide

Heterogeneity (I²)

I² ValueInterpretationAction
0-25%Low heterogeneityResults likely robust
25-50%Moderate heterogeneityConsider subgroup analysis
50-75%Substantial heterogeneityInvestigate sources, use random-effects
>75%Considerable heterogeneityQuestion pooling, explore meta-regression

CINeMA Confidence Ratings

  • High: Very confident the true effect is close to the estimate
  • Moderate: Moderately confident; true effect likely close to estimate
  • Low: Limited confidence; true effect may differ substantially
  • Very Low: Very little confidence; true effect likely differs substantially

Publication Bias Indicators

  • Egger's test p < 0.1: Suggests small-study effects (possible publication bias)
  • Asymmetric funnel plot: Visual indicator of potential bias
  • Trim and fill: Estimates number of missing studies

Node-Splitting Consistency

If p < 0.05 for any comparison, there may be inconsistency between direct and indirect evidence. Investigate potential effect modifiers.

Glossary of Terms

t² (tau-squared)
Between-study variance in random-effects models. Higher values indicate greater heterogeneity.
I²
Percentage of variability due to heterogeneity rather than sampling error. Ranges 0-100%.
Q statistic
Cochran's Q tests the null hypothesis of homogeneity. Significant Q suggests heterogeneity.
P-score
Probability that a treatment is better than another, averaged across all comparisons (0-1 scale).
SUCRA
Surface Under Cumulative Ranking curve. Equivalent to P-score × 100.
League Table
Matrix showing all pairwise treatment comparisons with effect estimates and confidence intervals.
Node-splitting
Method to assess consistency by separating direct and indirect evidence for each comparison.
CINeMA
Confidence In Network Meta-Analysis framework for assessing certainty of evidence.
GRADE
Grading of Recommendations Assessment, Development and Evaluation - quality assessment framework.
NNT
Number Needed to Treat. Derived from absolute risk difference.
Funnel Plot
Scatter plot of effect estimates vs precision. Asymmetry suggests publication bias.
Forest Plot
Visual display of treatment effects with confidence intervals.
